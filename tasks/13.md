Execute os seguintes passos rigorosamente de acordo com nossas .claude/rules e usando a stack atual (Drizzle, Next.js App Router, Zod, React Hook Form):

Lógica da Lista de Compras (actions/event-actions.ts):

Crie uma nova server action estritamente tipada chamada getEventShoppingList(eventId: number).

Esta função deve:
A. Buscar o evento, seu totalDrinks, e a proporção de cada bebida através de event_drinks.
B. Calcular a quantidade total de cada drink necessário para o evento (totalDrinks / quantidade de tipos de drinks escolhidos).
C. Iterar sobre os drinkIngredients de cada bebida, multiplicando a quantidade da receita pelo total daquela bebida no evento.
D. Agrupar tudo por Ingrediente para ter o "Total Necessário (em recipeUnit)".
E. Para cada ingrediente agrupado, calcular a Quantidade a Comprar: Math.ceil(Total Necessário / yieldQuantity).
F. Calcular o Custo Estimado: Quantidade a Comprar \* purchaseCost.

A função deve retornar um array de objetos contendo: nome do ingrediente, quantidade necessária (com recipeUnit), quantidade a comprar (com purchaseUnit) e custo estimado.
