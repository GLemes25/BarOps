"Crie o arquivo db/schema.ts. Precisamos criar o schema do Drizzle em PostgreSQL para o sistema de orçamentos. Crie as seguintes tabelas e suas relações:

- events: id, name, date, guests (inteiro), duration_hours (inteiro), avg_drinks_per_person (decimal), total_drinks (inteiro).

- drinks: id, name (ex: Caipirinha, Moscow Mule).

- ingredients: id, name, unit (ex: ml, un, garrafa), cost_per_unit (decimal). Inclua os ingredientes base e os compostos (ex: Xarope de açúcar, Suco de limão).

- drink_ingredients: Tabela pivô (many-to-many) ligando drinks e ingredients, contendo a quantidade daquele ingrediente para fazer 1 drink.

- event_labor: id, event_id (fk), role (ex: Bartender, Garçom), quantity (inteiro), cost_per_person (decimal).

- event_materials: id, event_id (fk), name (ex: Copo locado, Balcão), quantity (inteiro), cost_per_unit (decimal).

Exporte os schemas. Crie também o arquivo drizzle.config.ts na raiz para configurar o Drizzle Kit apontando para esse schema."
