"Crie uma pasta actions dentro de src. Crie um arquivo event-actions.ts.
Escreva Server Actions estritamente tipadas com TypeScript seguindo as regras em .claude/rules/typescript.mdx para:

createEvent: Recebe os dados do evento, insere no banco e retorna o ID.

getEventById: Busca um evento com todas as suas relações (mão de obra e materiais).

calculateEventCosts: Uma função complexa que recebe o ID de um evento. Ela deve:

Calcular total_drinks = guests \* avg_drinks_per_person.

Calcular o custo total de mão de obra (somando a tabela event_labor).

Calcular o custo total de materiais (somando event_materials).

Lógica de Espuma: Crie uma constante ou função auxiliar que saiba que 'Espuma' consome (Xarope de açúcar, Xarope de gengibre, Emulsificante e Suco de limão).
Sempre use try/catch e retorne objetos com { success: boolean, data?: any, error?: string }."
