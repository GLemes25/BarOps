"Precisamos evoluir o CRUD de Bebidas (app/drinks) para gerenciar as receitas usando a tabela pivô drink_ingredients. O usuário deve poder adicionar múltiplos ingredientes dinamicamente ao criar ou editar um drink.

Siga os passos abaixo rigorosamente de acordo com nossas .claude/rules (shadcn, react-hook-form, zod, typescript):

Server Actions (src/actions/drink-actions.ts):

Atualize getDrinks para trazer as relações: use a query do Drizzle para incluir os drinkIngredients e seus respectivos ingredients (usando with ou joins) para sabermos o que compõe o drink.

Atualize createDrink: Deve receber { name: string, ingredients: { ingredientId: number, quantity: number }[] }. Inicie uma transação (db.transaction). Insira o drink, pegue o id retornado e faça um insert múltiplo na tabela drink_ingredients.

Atualize updateDrink: Faça uma transação. Atualize o nome do drink. Para atualizar os ingredientes de forma simples, delete todos os registros em drink_ingredients onde drinkId for o id atual, e em seguida faça um novo insert múltiplo com os dados recebidos.

Tipagens e Schemas (app/drinks/drink-form.tsx e app/drinks/drinks-table.tsx):

Crie/Exporte um tipo IngredientOption = { id: number, name: string, unit: string }.

Atualize o DrinkRecord na tabela para incluir a lista de ingredientes que o compõem (ex: ingredients: { id: number, name: string, quantity: number, unit: string }[]).

Atualize o drinkSchema no formulário para: name: z.string(), e ingredients: z.array(z.object({ ingredientId: z.coerce.number(), quantity: z.coerce.number().positive() })).

O Formulário (app/drinks/drink-form.tsx):

O componente deve receber uma nova prop: availableIngredients: IngredientOption[].

Utilize o hook useFieldArray do react-hook-form referenciando o campo ingredients.

Na UI do formulário, abaixo do nome da bebida, crie um bloco para a receita. Para cada item do useFieldArray, renderize:

Um componente Select do shadcn para escolher o ingrediente (o value deve ser o ingredientId em string convertido para number). Mostre o name e a unit no label do select.

Um Input de número para a quantity.

Um Button (variante ghost ou destructive, com ícone de lixeira do lucide-react) para remover a linha.

Adicione um botão 'Adicionar Ingrediente' (variante outline) que faz o append de uma linha vazia no useFieldArray.

A Tabela e a Página (app/drinks/drinks-table.tsx e app/drinks/page.tsx):

Em page.tsx, além de chamar getDrinks(), importe e chame getIngredients() (da action de ingredientes) para pegar a lista do estoque.

Passe as duas listas para o <DrinksTable initialData={drinks} availableIngredients={ingredients} />.

Na DrinksTable, passe o availableIngredients para dentro do DrinkForm.

Na tabela visual, adicione uma nova coluna chamada 'Receita' e faça um map simples dos ingredientes do drink separados por vírgula (Ex: 'Vodka (50ml), Limão (20ml)').

Instale o componente select do shadcn se ainda não estiver instalado (pnpm dlx shadcn-ui@latest add select). Não inclua comentários no código gerado."
