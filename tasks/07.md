"Crie a estrutura de pastas e arquivos no App Router para as seguintes rotas: /events, /drinks, /ingredients, /labor, e /materials.

Para CADA uma dessas rotas:

Crie o page.tsx correspondente renderizando o layout com Cabeçalho, Tabela e Botão 'Novo'.

Na pasta components/forms/, crie o componente de formulário correspondente (ex: DrinkForm.tsx, IngredientForm.tsx). Cada formulário deve usar Zod para validação e o componente @components/ui/form do shadcn.

Integre os formulários dentro do Dialog nas respectivas páginas para as ações de Criação e Edição.

Na tabela, use dados estáticos iniciais ou arrays vazios apenas para montar o visual, mas prepare o evento onClick do botão de Excluir e do submit do Formulário para futuramente chamar as Server Actions."
