"Crie a rota dinâmica app/events/[id]/page.tsx. Siga as regras de .claude/rules.

Instale os componentes do shadcn: pnpm dlx shadcn-ui@latest add tabs badge card progress.

O page.tsx deve ser um Server Component assíncrono que recebe o id via params.

Busque os dados básicos do evento no banco usando a action getEventById(id).

Use o componente Tabs para criar quatro seções: 'Drinks', 'Equipe', 'Materiais' e 'Resumo Financeiro'.

Para cada aba, crie um componente cliente co-localizado na mesma pasta (ex: event-drinks-section.tsx, event-labor-section.tsx, etc.).

No cabeçalho da página, use o dayjs para exibir a data do evento formatada e um Badge mostrando o número de convidados."
