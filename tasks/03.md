"Crie a interface de criação de evento.

- Use a CLI do shadcn para adicionar os componentes necessários: pnpm dlx shadcn-ui@latest add form input button card separator.

- Em components/, crie o arquivo event-form.tsx.

- Crie um schema do Zod para os dados básicos do evento (nome, data, convidados, horas, drinks médios).

- Construa o formulário usando o @components/ui/form do shadcn, integrado com React Hook Form.

- Ao fazer o submit, chame a Server Action createEvent. Se der certo, redirecione o usuário (usando next/navigation) para a página de detalhes desse evento (ex: /events/[id])."
